
# Projeto de Planejamento de Casamento

Sistema web para organização e planejamento de casamento, com foco em controle de informações, centralização de dados e apoio à tomada de decisões durante a organização do evento.

O projeto tem como objetivo facilitar a gestão de convidados, presentes, buffet, fornecedores e planejamentos, oferecendo uma plataforma simples, funcional e organizada.

---

## Descrição

Aplicação desenvolvida para auxiliar no planejamento de casamentos, permitindo o gerenciamento completo das informações do evento em um único sistema.

O sistema contempla módulos de organização, controle e planejamento, ajudando o casal a manter todas as informações estruturadas e acessíveis.

---

## Funcionalidades

* Cadastro e gerenciamento de convidados
* Controle de confirmação de presença
* Organização por grupos e famílias
* Lista de presentes
* Controle de status dos presentes
* Cadastro de buffet e fornecedores
* Planejamento de cardápio
* Controle de restrições alimentares
* Cronograma do evento
* Checklist de tarefas
* Planejamento de etapas
* Controle de orçamento
* Organização de despesas

---

## Tecnologias Utilizadas

Frontend:

* Vue 3
* Quasar Framework v2
* TypeScript
* SASS

Backend:

* Node.js
* TypeScript

Banco de dados:

* SQLite
